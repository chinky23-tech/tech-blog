DOM Manipulation Basics

JAVASCRIPT August 10, 2024

The Document Object Model (DOM) is the programming interface for web documents. Learning to manipulate it with JavaScript was my gateway to creating dynamic, interactive web pages. Here's what I discovered.

What is the DOM?

The DOM is a tree-like representation of your webpage that JavaScript can interact with. When the browser loads your page, it creates this structure that we can modify.

DOM Tree Visualization

Every element, attribute, and piece of text becomes a node in this tree that we can access and modify.

Basic DOM Manipulation Methods

Selecting Elements

// Select single element
const header = document.querySelector('header');
// Select multiple elements
const buttons = document.querySelectorAll('.btn');

// Older methods (still useful)
const byId = document.getElementById('main');
const byClass = document.getElementsByClassName('item');
const byTag = document.getElementsByTagName('div');

Modifying Elements

// Change content
element.textContent = 'New text';
element.innerHTML = '<strong>Bold text</strong>';

// Change attributes
element.setAttribute('id', 'newId');
element.classList.add('active');
element.style.color = 'blue';

Creating and Removing Elements

Action Method Example
Create Element document.createElement() const div = document.createElement('div');
Add to DOM appendChild() or insertBefore() parent.appendChild(div);
Remove Element remove() or removeChild() element.remove();
Clone Element cloneNode() const newElement = element.cloneNode(true);

Event Listeners

The real power comes when you combine DOM manipulation with event listeners:

const button = document.querySelector('#myButton');

         button.addEventListener('click', function() {
           // DOM manipulation here
           const newItem = document.createElement('li');
             newItem.textContent = 'New item added!';
             document.querySelector('ul').appendChild(newItem);
           });

DOM Cheat Sheet

  • element.textContent - Get/set text content
  • element.innerHTML - Get/set HTML content
  • element.getAttribute() - Get attribute value
  • element.setAttribute() - Set attribute value
  • element.classList - Add/remove/toggle classes
  • element.style - Modify CSS styles
