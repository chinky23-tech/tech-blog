My First Frontend Project

PROJECTS July 5, 2024

After mastering HTML, CSS, and JavaScript basics, I built my first complete frontend project - a fully responsive interactive weather dashboard. Here's how I did it using only client-side technologies.

WeatherWise Dashboard

A responsive weather application that:

  • Fetches data from a public weather API
  • Saves search history in localStorage
  • Features animated weather icons
  • Works perfectly on all devices
Live Demo View Code

Key Features

API Integration

Used fetch() to get data from OpenWeatherMap API with async/await

Dynamic UI

DOM manipulation to update forecasts without page reload

Local Storage

Persisted search history between sessions

Mobile First

Designed for small screens then enhanced for desktop

Code Snippets

Fetching Weather Data

async function getWeatherData(city) {
    const response = await fetch(
        `https://api.openweathermap.org/data/2.5/weather?q=${city}&units=metric&appid=YOUR_API_KEY`
    );
    return await response.json();
}

LocalStorage Implementation

// Save searched city
function saveToHistory(city) {
    let history = JSON.parse(localStorage.getItem('weatherHistory')) || [];
    if (!history.includes(city)) {
        history.push(city);
        localStorage.setItem('weatherHistory', JSON.stringify(history));
    }
}

Deployment to GitHub Pages

  1. Created new GitHub repository named "weather-app"
  2. Initialized git locally: git init
  3. Added files and committed: git add . && git commit -m "Initial commit"
  4. Set remote: git remote add origin https://github.com/yourusername/weather-app.git
  5. Pushed code: git push -u origin main
  6. Enabled GitHub Pages in repository settings (Branch: main, Folder: /root)

My site went live at: https://yourusername.github.io/weather-app

Lessons Learned

  • API Keys: Learned to hide them from public repos (used a config.js file added to .gitignore)
  • Error Handling: Added try/catch blocks for API calls
  • Responsive Images: Used picture element with different sources
  • Git Workflow: Made small, frequent commits with clear messages
