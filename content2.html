CSS Flexbox Magic

CSS May 15, 2024

Flexbox revolutionized how I create layouts - goodbye float nightmares, hello intuitive alignment! Here's my journey from struggling with positioning to mastering this game-changing CSS module.

The Before Times

My early attempts with floats and clearfix hacks:

/* The old way */
     .container {
    overflow: hidden; /* Clearfix hack */
 }
     .box {
    float: left;
    width: 30%;
    margin-right: 5%;
}
.box:last-child {
    margin-right: 0;
}

Flexbox Enlightenment

The moment I discovered these core properties:

  • display: flex - The magic incantation
  • justify-content - Main axis alignment
  • align-items - Cross axis alignment
  • flex-direction - Row/column flow
  • flex-grow/shrink - Responsive control

My First Flexbox Layout

.card-container {
    display: flex;
    gap: 20px;
    justify-content: center;
    flex-wrap: wrap;
}
.card {
    flex: 1 1 300px; /* Grow | Shrink | Basis */
}

Pro Tips I Learned

  • Use gap instead of margins between items
  • flex: 1 is shorthand for equal distribution
  • Combine with min-width for responsive grids
  • Remember align-self for individual items

Stumbling Blocks

  • Forgetting to set parent container as flex
  • Confusing main vs cross axis alignment
  • Overusing fixed widths that break flexibility
  • Browser prefix requirements for older sites
